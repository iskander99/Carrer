Поддержка - новое сообщение

Точка входа: api/v1/cabinet/support/message 




Пример запроса:

{
    "appeal_id" : 1,
    "message" : "Сообщение"
}

Успешный ответ - код 200:

{
    "success": "Сообщение успешно отправлено."
}

Неудачный ответ - код 404:

{
    "errors": {
        "user" : "Пользователь не найден."
    }
}

POST запрос

Параметр Тип Описание
appeal_id integer ID обращения
message string (max 10 000) Комментарий
