POST запрос

Параметр Тип Описание
role integer Роль: 1 - Кандидат, 2 - Консультант
tag_rule integer Тэг
first_name string Имя
last_name string Фамилия
phone string Номер телефона
email string Почта
birth_day string Дата рождения
citizenship string Гражданство
region integer Регион проживания
city string Город
prof_field integer Профессиональная область
desired_post string Желаемая должность
desired_salary integer Ожидаемый доход
search_region integer Регион поиска
search_industry integer Отрасль поиска
edu_list array Список: Образование
level integer Уровень образования
institution string Учебное заведение
faculty string Факультет
specialization string Специальность
edu_years_from integer Годы обучения от
edu_years_to integer Годы обучения до
exp_list array Список: Опыт работы
organization string Организация
exp_mode integer Самозанятый, ИП ...
post string Должность
duties string Обязанности
exp_years_from integer Годы работы от
exp_years_to integer Годы работы до
now integer По настоящее время
password string Пароль
password_confirmation string Подтверждение пароля
img file Изображение профиля
